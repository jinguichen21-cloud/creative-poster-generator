# 创意海报生成

根据自然语言描述自动生成高质量创意海报，支持多种风格和场景。

## 概述

本技能提供智能化的海报生成功能：
- 根据文字描述自动生成创意海报
- 支持多种海报风格（科技、商务、活动、节日等）
- 支持多种应用场景（产品发布、会议邀请、促销宣传等）
- 生成的海报可直接下载使用

## 前置准备

本技能需要创意海报生成 MCP 服务，通过环境变量注入。

| 环境变量 | MCP 服务 | mcpId | 获取链接 |
|----------|----------|-------|----------|
| `$POSTER_URL` | 创意海报生成 | 9436 | [获取 URL](https://mcp.dingtalk.com/#/detail?mcpId=9436&detailType=marketMcpDetail) |

**配置说明**：
1. `mcp-config.json` 中已配置 mcpId，URL 字段由市场链路根据 mcpId 自动生成
2. 本地开发测试时，可手动填入从钉钉 MCP 市场获取的 StreamableHTTP URL
3. `mcp-config.json` 不会打包进 `.skill`，请勿提交到代码仓库

详细说明见 [mcp-server-config-guide.md](references/mcp-server-config-guide.md)

## 工作流程

### 场景 1：根据描述生成海报

1. 用户提供海报描述
   - 海报主题
   - 风格要求
   - 使用场景
   - 文字内容（可选）

2. 优化提示词
   - 分析用户需求
   - 补充细节描述
   - 构建完整的生成提示

3. 调用 `generation_poster_by_text` 生成海报
   - 传入优化后的提示词
   - 等待生成结果

4. 返回生成结果
   - 海报图片 URL
   - 下载链接
   - 预览展示

### 场景 2：交互式海报设计

1. 询问用户海报用途和风格偏好
2. 引导用户提供关键信息
3. 自动生成提示词并生成海报
4. 根据反馈迭代优化

## 海报风格指南

### 支持的风格类型

| 风格 | 描述 | 适用场景 |
|------|------|----------|
| **科技风** | 蓝色调、几何图形、未来感 | 产品发布、技术大会、AI主题 |
| **商务风** | 简洁大气、专业配色、高端感 | 企业宣传、商务会议、品牌展示 |
| **活动风** | 色彩丰富、动感元素、吸引力 | 促销活动、节日庆典、线下活动 |
| **文艺风** | 柔和色调、手绘元素、温馨感 | 读书分享、艺术展览、文化活动 |
| **卡通风** | 可爱元素、明亮色彩、趣味性 | 儿童活动、亲子活动、轻松主题 |
| **国潮风** | 中国传统元素、红色金色、古典美 | 传统节日、国风活动、文化宣传 |

## 工具说明

### generation_poster_by_text
根据自然语言生成高质量海报。

**参数**：
- `text` (string, 必填): 海报描述文本，建议详细描述主题、风格、元素、色调等

**返回值**：
- `image_url`: 生成的海报图片 URL
- `task_id`: 任务ID（用于追踪）

**提示词示例**：

1. **产品发布会**
   ```
   生成一张科技风格的产品发布会海报，主题是"AI智能助手新品发布"。
   主标题：AI智能助手 Pro
   副标题：开启智能办公新时代
   色调：深蓝色配紫色渐变
   元素：科技感背景、AI芯片图案、光束效果
   氛围：未来感、专业、高端
   ```

2. **会议邀请**
   ```
   生成一张商务风格的会议邀请函海报，主题是"2026年度战略研讨会"。
   主标题：战略研讨会
   副标题：聚势共赢 · 智创未来
   色调：金色配深灰色
   元素：城市天际线、几何线条、商务人士剪影
   氛围：正式、高端、专业
   ```

3. **促销活动**
   ```
   生成一张活动风格的促销海报，主题是"双11狂欢节"。
   主标题：双11狂欢节
   副标题：全场5折起 · 限时抢购
   色调：红色配橙色
   元素：购物车、礼物盒、烟花、折扣标签
   氛围：热闹、喜庆、紧迫感
   ```

## 调用示例

```bash
# 生成海报
python3 scripts/call_mcp.py call "$POSTER_URL" generation_poster_by_text \
  --params '{"text": "生成一张科技风格的产品发布会海报，主题是AI智能助手新品发布，主标题：AI智能助手 Pro，副标题：开启智能办公新时代，色调：深蓝色配紫色渐变，元素：科技感背景、AI芯片图案、光束效果"}'
```

## 最佳实践

### 提示词优化建议

1. **具体描述**
   - ✅ 好："蓝色科技感背景，带有数据流和光效"
   - ❌ 差："好看的背景"

2. **明确风格**
   - ✅ 好："商务简约风格，白色背景配金色文字"
   - ❌ 差："做个海报"

3. **包含关键元素**
   - 主标题（必填）
   - 副标题（可选）
   - 色调偏好
   - 视觉元素
   - 整体氛围

4. **避免敏感内容**
   - 不包含政治敏感内容
   - 不包含暴力、色情内容
   - 不包含侵权元素

## 异常处理

- **生成超时** → 提示用户网络繁忙，建议稍后重试
- **提示词敏感** → 提示用户修改描述，避免敏感内容
- **生成质量不佳** → 建议用户增加更多细节描述，或更换风格
- **服务不可用** → 提示用户检查 MCP 配置或联系管理员
